# Indiciário

Indiciário é um gerador de jogos de investigação offline em formato de dossiê com envelopes. O projeto combina blueprints estruturados, validação narrativa e renderização em PDF para produzir casos jogáveis sem dependência de aplicativos externos.

## Caso canônico atual

O caso canônico do repositório é **“O Desvio da Reserva Mirante”**, atualmente classificado como **Iniciante** por decisão editorial de produto.

Arquivo atual:

- `examples/caso_canonico_iniciante.json`

### Roadmap de nomenclatura

O arquivo acima mantém temporariamente o nome histórico `caso_canonico_iniciante.json` para preservar compatibilidade com testes, scripts e referências existentes. Em uma próxima etapa, ele deverá ser renomeado para:

- `examples/caso_canonico_iniciante.json`

Quando a renomeação for feita, todos os testes, comandos documentados e entrypoints que referenciam o nome antigo deverão ser atualizados juntos.
