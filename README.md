# Indiciário

Indiciário é um gerador de jogos de investigação offline em formato de dossiê com envelopes. O projeto combina blueprints estruturados, validação narrativa e renderização em PDF para produzir casos jogáveis sem dependência de aplicativos externos.

## Caso canônico atual

O caso canônico do repositório é **“O Desvio da Reserva Mirante”**, atualmente classificado como **Iniciante** por decisão editorial de produto.

Arquivo atual:

- `examples/caso_canonico_iniciante.json`

Use este arquivo como referência principal em testes, scripts e comandos de validação canônica.
